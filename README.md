
# ecommerce-admin

Pre-requirement

- python (prefer python3)
- pip (tool to install package)


## Step

- Create Virtual Env

  ```
  python -m venv env
  . env/bin/activate
  pip install Django
  ```

- install all dependencies
  ```
  pip install -r requirements.txt
  ```

- Start app with command:
  ```
  ./manage.py runserver
  ```
